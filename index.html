WHAT IS A NEURAL NETWORK?

THE BIOLOGICAL METAPHOR

A neural network is a layered graph of computational units (neurons). Each neuron takes weighted inputs, sums them with a bias, then passes the result through a non-linear activation function. Stacking many layers allows the network to learn arbitrarily complex functions.

LAYERS

Input layer: raw features — one node per feature.
Hidden layers: learned representations. More layers = more abstraction.
Output layer: one node per class (softmax) or one node for regression.

FORWARD PASS

For layer l, neuron j: z[l][j] = Σᵢ w[l][i][j]·a[l-1][i] + b[l][j] a[l][j] = f( z[l][j] ) f = activation function (ReLU, sigmoid…) Output: final layer activations = predictions

BACKPROPAGATION

Loss L → compute ∂L/∂w via chain rule: δ[L] = ∂L/∂z[L] (output error) δ[l] = (W[l+1]ᵀ δ[l+1]) ⊙ f'(z[l]) ∂L/∂W[l] = δ[l] · a[l-1]ᵀ Update: w ← w − α·∂L/∂w
KEY CONCEPTS

UNIVERSAL APPROXIMATION

A network with even one hidden layerHidden Layer
A layer between input and output. Learns intermediate feature representations. The "deep" in deep learning refers to having many hidden layers.Wikipedia ↗ and enough neurons can approximate any continuous function on a compact domain. Depth (more layers) is usually more efficient than width (more neurons per layer) for complex tasks.

OVERFITTING & REGULARISATION

Deep networks can memorise training data. Remedies: dropoutDropout
Randomly sets a fraction of neurons to zero during each training step. Forces the network to learn redundant representations, reducing overfitting.Wikipedia ↗ (randomly zero neurons), L2 weight decayL2 Regularisation (Weight Decay)
Adds λΣw² to the loss. Penalises large weights, encouraging simpler models. Equivalent to a Gaussian prior on weights.Wikipedia ↗, early stoppingEarly Stopping
Halt training when validation loss stops improving. Simple but effective. Prevents the model from memorising noise in the training set.Wikipedia ↗, and batch normalisationBatch Normalisation
Normalises layer inputs per mini-batch. Stabilises training, allows higher learning rates, and reduces sensitivity to weight initialisation.Wikipedia ↗.

VANISHING GRADIENTS

SigmoidSigmoid
σ(z) = 1/(1+e⁻ᶻ). Output ∈ (0,1). Gradient ≤ 0.25, so after L layers gradient shrinks by 0.25ᴸ → nearly zero for deep networks.Wikipedia ↗/tanh saturate — gradients shrink to near-zero as they propagate back through many layers. Solution: use ReLUReLU (Rectified Linear Unit)
f(z) = max(0,z). Gradient = 1 for z>0, so no shrinkage. Fast, simple, and the default choice for hidden layers.Wikipedia ↗ activations, residual connectionsResidual Connections (Skip Connections)
Add the input of a layer directly to its output: y = F(x) + x. Allows gradients to flow straight back through the skip path, enabling very deep networks.Wikipedia ↗, or careful weight initialisation (He, Xavier).

OPTIMISERS

Beyond vanilla gradient descentGradient Descent
Iteratively update weights in the direction that reduces loss: w ← w − α·∂L/∂w. The engine behind all neural network training.Wikipedia ↗: MomentumMomentum
Accumulates a velocity vector in the gradient direction. Helps escape shallow local minima and speeds up convergence in consistent gradient directions.Wikipedia ↗ accumulates velocity, RMSPropRMSProp
Scales learning rate by a running average of squared gradients. Adapts per-parameter, reducing oscillation in noisy gradient directions.Wikipedia ↗ scales by gradient variance, AdamAdam (Adaptive Moment Estimation)
Combines momentum and RMSProp. Maintains both first and second moment estimates. Most popular optimiser in modern deep learning.Wikipedia ↗ combines both and is the most common modern choice.

CHAIN RULE — STEP BY STEP

LOSS → OUTPUT

Cross-entropy: L = −Σ y log(a_out) ∂L/∂a_out = a_out − y (for softmax+CE) ∂L/∂z_out = ∂L/∂a_out · f'(z_out)

HIDDEN ← OUTPUT

Error propagates backwards: δ[l] = (W[l+1]ᵀ · δ[l+1]) ⊙ f'(z[l]) ⊙ = element-wise multiply f'(z) is why activation choice matters!

WEIGHT GRADIENTS

∂L/∂W[l] = δ[l] · a[l-1]ᵀ ∂L/∂b[l] = δ[l] W[l] ← W[l] − α · ∂L/∂W[l] Larger δ → larger weight update

VANISHING GRADIENT

Sigmoid: f'(z) ≤ 0.25 After L layers: gradient ≤ 0.25ᴸ L=10: ≤ 9.5e-7 → near zero! ReLU: f'(z) = 1 for z>0 → no shrink
